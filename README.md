# Koyeb-keepalive

使用 Github Actions 以让 Koyeb 保持账户活跃

## 部署说明

* Fork 这个项目，然后启动
* 项目用到的环境变量
  | 变量名 | 是否必须 | 默认值 | 备注 |
  | ------------ | ------ | ------ | ------ |
  | API_URLS | 是 | 无 | 应用url'xxx.koyeb.app' or 'your_argo_domain'用,分隔多应用 |
  | APP_NAMES | 是 | 无 | app名，用,分隔多应用|
  | API_KEYS | 是 | 无 | 用户access_token,到用户设置生成
